## It was written in native rust code with some rust libraries like:

## [Crossterm](https://docs.rs/crossterm) - deal with terminal.

## [Clap](https://docs.rs/clap) - args parsing.

## [Syntect](https://docs.rs/syntect) - simple tool to highlight syntax depending on what file i've open.

## [Shellexpand](https://docs.rs/shellexapnd) - tool that process relative paths to absolute ~/ -> /home/username/

## [once_cell](https://docs.rs/once_cell) - to avoid some problems with lifetimes
